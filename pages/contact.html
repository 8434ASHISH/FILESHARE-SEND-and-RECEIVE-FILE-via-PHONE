FAQ

Is my data safe?

Yes! FileShare P2P uses end-to-end encryption via WebRTC. Files are transferred directly between peers.

How long do files stay on your servers?

Files are not stored on our servers. They're transferred directly between peers. It's a true P2P system.

What's the file size limit?

We support files up to 5GB. For larger files, we recommend splitting into multiple transfers.

Can I use this on mobile?

Yes! FileShare P2P works on all mobile browsers. Install as a progressive web app for quick access.
